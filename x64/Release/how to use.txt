📘 How to Use yt-dlp GUI Wrapper
⚙️ First-Time Setup (IMPORTANT)

Before using the application, you must install required dependencies.

👉 Step 1: Run the setup script as Administrator

Right-click the file:

setup.ps1

Then select:

Run with PowerShell (Run as Administrator)

This will automatically install:

yt-dlp
ffmpeg
Chocolatey (if not already installed)
⚠️ If you skip this step

The application will not be able to:

Download videos
Convert audio (MP3, M4A, Opus)
Merge video streams
🚀 Step 2: Launch the application

After setup completes, simply run:

yt-dlp GUI.exe

No additional configuration is required if installation succeeded.

🔧 Optional (Manual Mode)

If yt-dlp is not detected automatically, you can:

Place yt-dlp.exe in the same folder as the app
Or specify its path inside the UI
📦 Requirements
Windows 10/11
Internet connection (for downloads)
ffmpeg (installed automatically via setup script)
💡 Notes
Downloads are handled entirely by yt-dlp
This GUI only builds and executes commands
You can safely delete the setup script after installation
